FSMO Roles

Tutorial – Active Directory Health Check

Objective

This check verifies that all FSMO roles are present and assigned to active Domain Controllers.

Prerequisites

  • Access to a Domain Controller
  • Administrator permissions

Check FSMO Roles

Open Command Prompt and run:

CMD 
netdom query fsmo

Verify that:

  • All 5 FSMO roles are displayed
  • The listed Domain Controllers are online
  • No errors are returned

Result Interpretation

  • ✅ OK – All FSMO roles are present and accessible.
  • ⚠️ WARNING – FSMO roles are present but hosted on unstable Domain Controllers.
  • ❌ NOK – Missing FSMO roles or unreachable role holders.
