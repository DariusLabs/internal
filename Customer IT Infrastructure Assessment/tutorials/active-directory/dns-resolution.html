DNS Resolution

Tutorial – Active Directory Health Check

Objective

This check verifies that DNS resolution is working correctly in the Active Directory environment.

Prerequisites

  • Access to a domain-joined machine
  • Basic user permissions

Test DNS Resolution

Open Command Prompt and run:

CMD 
nslookup domain.local

Verify that:

  • An IP address is returned
  • No timeout or DNS errors appear

Verify DNS Configuration

Run:

CMD 
ipconfig /all

Verify that:

  • DNS servers point to internal Domain Controllers
  • No public DNS servers are configured

Result Interpretation

  • ✅ OK – DNS resolution is working correctly.
  • ⚠️ WARNING – Minor DNS configuration or resolution issues detected.
  • ❌ NOK – DNS resolution failures detected.
