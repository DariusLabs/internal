SYSVOL & NETLOGON

Tutorial – Active Directory Health Check

Objective

This check verifies that SYSVOL and NETLOGON are available and correctly replicated across Domain Controllers.

Prerequisites

  • Access to a Domain Controller
  • Administrator permissions

Verify SYSVOL & NETLOGON Shares

Open Command Prompt and run:

CMD 
net share

Verify that:

  • SYSVOL share is present
  • NETLOGON share is present
  • No missing or inaccessible shares

Check DFSR Status

Run:

CMD 
wmic /namespace:\\root\microsoftdfs path dfsrreplicatedfolderinfo get replicatedfoldername,state

Verify that:

  • SYSVOL state is 4 (Normal)
  • No replicated folder is in error state
  • No state 5 (Error) is displayed

Result Interpretation

  • ✅ OK – SYSVOL and NETLOGON are available and DFSR replication is healthy.
  • ⚠️ WARNING – Minor DFSR delays or replication inconsistencies detected.
  • ❌ NOK – Missing shares or DFSR replication failures detected.
