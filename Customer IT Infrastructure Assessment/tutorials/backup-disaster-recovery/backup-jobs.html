Backup Jobs

Tutorial – Backup & Disaster Recovery

Objective

This check verifies that backup jobs are properly configured and operational.

Backup job issues can impact:

  • Data protection
  • Disaster recovery
  • Business continuity
  • Restore capabilities

Prerequisites

  • Access to the backup console
  • User account with read or admin permissions
  • Access to backup infrastructure

Verify Backup Jobs

Open the backup console and navigate to:

Home → Jobs

Verify that:

  • Backup jobs are present
  • Jobs are enabled
  • No critical backup jobs are missing

Verify Backup Job Status

Check the latest execution status of the backup jobs.

Home → Last 24 Hours

Verify that:

  • Latest runs are successful
  • No failed jobs are present
  • No jobs are stuck or disabled

Result Interpretation

  • ✅ OK – Backup jobs are configured and running successfully.
  • ⚠️ WARNING – Backup jobs completed with warnings or partial issues.
  • ❌ NOK – Backup jobs failed, missing, or disabled.
