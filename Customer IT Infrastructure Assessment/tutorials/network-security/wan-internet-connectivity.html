WAN / Internet Connectivity

Tutorial – Network / Security

Objective

This check verifies that the server or workstation can successfully access external internet resources.

Prerequisites

  • Access to a Windows server or workstation
  • Command Prompt access

Verify Internet Connectivity Path

Open Command Prompt and run:

CMD 
tracert google.be

Verify that:

  • The trace successfully reaches external hosts
  • No major packet loss or timeout occurs
  • Internet routing is operational

Result Interpretation

  • ✅ OK – Internet connectivity and WAN routing are operational.
  • ⚠️ WARNING – Slow routing, latency, or intermittent connectivity detected.
  • ❌ NOK – No internet connectivity or routing failure detected.
