Audit Logs

Tutorial – Microsoft 365 Security

Objective

This check verifies that Microsoft 365 audit logging is enabled.

Missing audit logs can impact:

  • Security investigations
  • Incident response
  • User activity tracking
  • Compliance requirements

Prerequisites

  • Access to the Microsoft Purview Portal
  • User account with Compliance Admin or Global Admin permissions

Verify Audit Logging

Open the Microsoft Purview portal and navigate to:

Microsoft Purview → Solutions → Audit

Direct Link 
https://purview.microsoft.com/audit

Verify that:

  • Audit logging is enabled
  • No "Start recording user and admin activity" message is displayed
  • Audit events can be searched

Result Interpretation

  • ✅ OK – Audit logging is enabled and operational.
  • ⚠️ WARNING – Audit logging is partially configured or limited.
  • ❌ NOK – Audit logging is disabled or unavailable.
