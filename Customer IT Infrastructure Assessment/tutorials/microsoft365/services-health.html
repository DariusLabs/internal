Microsoft 365 Service Health

Tutorial – Microsoft 365 Security

Objective

This check verifies that Microsoft 365 core services are operational and healthy.

Microsoft 365 service issues can impact:

  • Email services
  • File access and synchronization
  • Microsoft Teams
  • User authentication
  • Business productivity

Prerequisites

  • Access to the Microsoft 365 Admin Center
  • User account with admin or reader permissions

Verify Microsoft 365 Service Health

Open the Microsoft 365 Admin Center and navigate to:

Microsoft 365 Admin Center → Health → Service Health

Direct Link 
https://admin.microsoft.com/Adminportal/Home#/servicehealth

Verify that core Microsoft 365 services are healthy:

  • Exchange Online
  • SharePoint Online
  • Microsoft Teams
  • OneDrive
  • Microsoft Entra ID

Verify that no major incidents or service degradations are present.

Result Interpretation

  • ✅ OK – Core Microsoft 365 services are healthy and operational.
  • ⚠️ WARNING – Minor service degradations or advisories detected.
  • ❌ NOK – Major incidents or service outages detected.
