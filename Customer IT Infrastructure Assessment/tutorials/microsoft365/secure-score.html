Microsoft 365 Secure Score

Tutorial – Microsoft 365 Security

Objective

This check verifies the overall Microsoft 365 security posture using Secure Score.

A low Secure Score may indicate:

  • Weak security configurations
  • Missing protection features
  • Identity security risks
  • Increased attack exposure

Prerequisites

  • Access to the Microsoft Defender Portal
  • User account with Security Reader or admin permissions

Verify Secure Score

Open the Microsoft Defender portal and navigate to:

Microsoft Defender → Exposure Management → Secure Score

Direct Link 
https://security.microsoft.com/securescore

Verify that:

  • The Secure Score is not critically low
  • No critical improvement actions are ignored
  • The score trend is stable or improving

Result Interpretation

  • ✅ OK – Secure Score above 50%.
  • ⚠️ WARNING – Secure Score between 30% and 50%.
  • ❌ NOK – Secure Score below 30%.
