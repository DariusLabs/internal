File-Level Restore

Tutorial – File Server Health Check

Objective

This check verifies that file-level restore is operational in the backup solution.

File-level restore allows recovery of:

  • Deleted files
  • Modified files
  • Folders or documents

Prerequisites

  • Access to the backup solution
  • User account with backup operator or admin permissions

Verify File-Level Restore

Open the Veeam console and navigate to:

  • Home
  • Restore
  • Restore from backup
  • Guest files restore
  • Microsoft Windows

Then:

  • Select the backup job
  • Select the virtual machine
  • Select a restore point

Verify that files and folders are visible in the restore browser.

Result Interpretation

  • ✅ OK – File-level restore operational and restore points available.
  • ⚠️ WARNING – Restore points available but verification incomplete.
  • ❌ NOK – File-level restore unavailable or backup inaccessible.
