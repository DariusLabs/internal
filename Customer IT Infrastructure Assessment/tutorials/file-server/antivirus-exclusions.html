Antivirus Exclusions

Tutorial – File Server Health Check

Objective

This check verifies that the antivirus is installed and properly configured on the File Server.

Incorrect antivirus configuration or missing exclusions can cause:

  • Slow file access
  • Locked files
  • Backup failures
  • Application instability

Prerequisites

  • Access to the File Server
  • User account with admin permissions
  • Access to the antivirus console

Verify Antivirus Exclusions

Open the antivirus console on the File Server.

Navigate to:

  • Configuration
  • Server

Verify that:

  • The antivirus protection is active
  • Automatic exclusions are enabled
  • No critical warning or protection issue is displayed

Automatic exclusions help prevent performance issues and file access problems on the File Server.

Result Interpretation

  • ✅ OK – Antivirus active and automatic exclusions enabled.
  • ⚠️ WARNING – Antivirus installed but exclusions disabled or incomplete.
  • ❌ NOK – Antivirus missing or protection disabled.
