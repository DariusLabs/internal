Storage Growth Trend

Tutorial – File Server Health Check

Objective

This check verifies the storage growth trend on the File Server.

Monitoring storage growth helps anticipate:

  • Disk saturation
  • Unexpected storage consumption
  • Future storage expansion needs

Prerequisites

  • Access to the File Server
  • User account with read or admin permissions
  • Access to disk usage monitoring or historical data

Verify Storage Usage Trend

Compare current disk usage with previous measurements or monitoring data.

Verify that:

  • Disk usage growth remains stable
  • No abnormal increase is detected
  • Sufficient free space remains available

Pay particular attention to:

  • File shares volumes
  • Backup volumes
  • User profiles folders

Result Interpretation

  • ✅ OK – Storage growth stable and under control.
  • ⚠️ WARNING – Rapid storage growth observed.
  • ❌ NOK – Storage capacity critically low or saturation imminent.
