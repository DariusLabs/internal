External DNS Resolution

Tutorial – Network / Security

Objective

This check verifies that external DNS resolution works correctly for internet access and public services.

External DNS issues can cause:

  • Internet browsing failures
  • Cloud service connectivity problems
  • Microsoft 365 access issues
  • Slow external network access

Prerequisites

  • Access to a Windows workstation or server
  • CMD or PowerShell access

Test External DNS Lookup

Open CMD or PowerShell.

Run the following command:

CMD 
nslookup google.be

Verify that:

  • A DNS server responds
  • An external IP address is returned
  • No timeout or resolution error appears

Result Interpretation

  • ✅ OK – External DNS resolution works correctly.
  • ⚠️ WARNING – Slow responses or intermittent DNS issues detected.
  • ❌ NOK – External DNS requests fail or timeout.

Reliable external DNS is required for internet and cloud connectivity.
