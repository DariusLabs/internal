Shadow Copies

Tutorial – File Server Health Check

Objective

This check verifies that Shadow Copies (Previous Versions) are enabled on the File Server volumes.

Shadow Copies allow quick recovery of:

  • Deleted files
  • Modified files
  • Previous folder versions

Prerequisites

  • Access to the File Server
  • User account with admin permissions

Verify Shadow Copies

On the File Server:

  • Open This PC
  • Right-click the data volume
  • Select Configure Shadow Copies

Verify that:

  • Shadow Copies are enabled on data volumes
  • A storage limit is configured
  • Scheduled snapshots are present

Result Interpretation

  • ✅ OK – Shadow Copies enabled and configured correctly.
  • ⚠️ WARNING – Shadow Copies enabled but schedule or storage configuration incomplete.
  • ❌ NOK – Shadow Copies disabled or missing on data volumes.
